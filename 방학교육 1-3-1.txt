프로그램/코드의 흐름
순차 (내려가는 경우)
반복 (반복되는 경우)
조건 (구간이 갈리는 경우)

순차 : 가장 기본적이고 쉬우면서 강력한 구조
내려가면서 순차적으로 실행한다. 
1번부터 10번라인까지 내려가면서 실행.

반복 : 
for 문의 구조
for i in 범위
반복할 명령어 1
반복할 명령어 2

while 문의 구조
while 조건 :
반복할 명령어 1
반복할 명령어 2

조건:
if 문의 구조
if (조건문) :
실행할 코드 1
실행할 코드 2

if elif else 구문 사용법 - if 를 거치든 elif를 거치든 else를 거치든 경로가 갈라짐.
if (조건문)
실행할 코드 1
실행할 코드 2
elif(조건문)
실행할 코드 3
실행할 코드 4
else(조건문)
실행할 코드 5
실행할 코드 6

순차, 반복, 조건을 조합하여 복잡한 흐름을 만들 수 있다.
